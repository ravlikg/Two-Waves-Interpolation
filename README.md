## 🚀 Швидкий запуск без компіляції (Завантаження .exe)

Якщо ви не хочете збирати проєкт із сирців, ви можете завантажити вже скомпільовану версію програми (виконавчий файл `.exe`) прямо з GitHub.

### Покрокова інструкція:

1. Перейдіть на головну сторінку цього репозиторію.
2. У правій колонці знайдіть блок **Releases** та натисніть на останню версію (вона позначена зеленим тегом `Latest`).

![Де знайти релізи на сторінці](./assets/download-guide1.png)
*Мал. 1 — Розташування блоку Releases у правій частині екрана*

3. У списку файлів, що відкриється (розділ **Assets**), завантажте архів `.zip` або напряму файл `interference.exe`.

![Завантаження EXE файлу](./assets/download-guide2.png)
*Мал. 2 — Завантаження виконавчого файлу з блоку Assets*

4. Розархівуйте завантажений архів (якщо завантажили `.zip`) та запустіть файл `interference.exe` подвійним кліком.

> ⚠️ **Важливо:** Для коректної роботи програми переконайтеся, що файл шейдера `wave.frag` лежить в тій самій папці, що і запущений `.exe` файл (або за шляхом, який вимагає програма), інакше OpenGL не зможе завантажити графічні ефекти.
